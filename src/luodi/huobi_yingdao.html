USDT支付引导

一. 下载与注册

请前往火币官网 huobi.de.com 下载APP，并完成注册和实名认证

二. 购买USDT币

在APP首页 => 快捷买币 => 我要买 => USDT，选择支付方式和数量进行筛选商家完成交易
温馨提醒：可以选择无"广告方已设限"标志的商家进行交易（有此标志交易需要高级视频认证）

三. 购买我们APP金币

1、复制收款页面的转账地址
2、在火币APP => 资产 => 提币
选择 ERC20，填入前一步复制的地址
数量处输入订单显示的USDT个数 + 手续费的总个数
3、确认信息无误后点击 提币，等待到账即可
举例：如我需要充值50元，在我们APP内选择50元下单，生成的网页中展示需要支付 7.79个 USDT币，手续费为5.00个USDT 在数量处则需要输入7.79 + 5 = 12.79
